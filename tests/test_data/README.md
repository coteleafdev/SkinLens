# Test Data Directory

This directory contains test data for integration and unit tests.

## Structure

- `images/` - Test images for analysis tests
- `cases/` - Test case JSON files for batch workflow tests
- `config/` - Test configuration files

## Adding Test Data

When adding new test data:
1. Use small, synthetic images when possible
2. For real images, ensure proper licensing
3. Document the source and purpose of each test file
