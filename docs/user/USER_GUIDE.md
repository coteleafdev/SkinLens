# 사용자 가이드 (User Guide)

> **문서 버전:** 1.0.0  
> **대상 프로젝트 버전:** 1.0.0  
> **마지막 업데이트:** 2026-05-31  
> **상태:** 활성

---

## 개요

SkinLens는 AI 기반 피부 분석 시스템으로, 원본 이미지와 복원 이미지를 비교하여 피부 상태를 정량적으로 분석합니다.

---

## 시작하기

### 시스템 요구사항

- **웹 브라우저:** Chrome, Firefox, Safari, Edge (최신 버전)
- **이미지 형식:** JPG, JPEG, PNG, WEBP
- **이미지 크기:** 최대 20MB
- **인터넷 연결:** 필수

---

## 사용 방법

### 1. 로그인

1. SkinLens 웹사이트 접속
2. 고객 ID와 비밀번호 입력
3. 로그인 버튼 클릭

### 2. 이미지 업로드

1. "새 분석" 버튼 클릭
2. 피부 이미지 선택 (정면 촬영 권장)
3. 업로드 버튼 클릭

**이미지 촬영 가이드:**
- 조명: 자연광 또는 균일한 인공조명
- 각도: 정면, 약 30cm 거리
- 표정: 무표정, 눈 뜬 상태
- 메이크업: 없거나 최소화

### 3. 분석 결과 확인

분석이 완료되면 다음 정보를 확인할 수 있습니다:

- **종합 점수:** 원본/복원 이미지의 종합 피부 점수 (0-100)
- **개별 항목 점수:** 18개 측정 항목별 점수
- **LLM 소견:** AI 기반 피부 상태 분석
- **제품 추천:** 피부 타입에 맞는 제품 추천

### 4. 피부 타입 확인

AI가 감지한 피부 타입을 확인하고 수정할 수 있습니다:

1. 분석 결과 화면에서 "피부 타입 확인" 버튼 클릭
2. 실제 피부 타입 선택 (건성, 지성, 복합성, 민감성)
3. 확인 버튼 클릭

## 점수 해석

### 종합 점수

- **90-100:** 매우 좋음
- **80-89:** 좋음
- **70-79:** 보통
- **60-69:** 개선 필요
- **0-59:** 개선이 시급함

### 개별 항목 점수

#### 색소 관련
- **기미 점수 (melasma_score):** 기미 농도
- **주근깨 점수 (freckle_score):** 주근깨 정도
- **색소침착 점수 (pigmentation_score):** 전체 색소침착

#### 모공/질감 관련
- **모공 점수 (pore_score):** 모공 크기/개수
- **거칠기 점수 (roughness_score):** 피부 표면 거칠기
- **결점 점수 (spot_score):** 잡티/색점 정도

#### 주름/탄력 관련
- **잔주름 점수 (fine_wrinkle_score):** 미세 주름
- **깊은주름 점수 (deep_wrinkle_score):** 깊은 주름
- **탄력 점수 (elasticity_score):** 피부 탄력

#### 기타
- **홍조 점수 (redness_score):** 피부 홍조
- **트러블 점수 (acne_score):** 트러블/여드름
- **트러블 후 색소 점수 (post_acne_pigment_score):** 트러블 후 색소침착

## 피부 타입

### 건성 (Dry)
- 피부가 건조하고 당김이 느껴짐
- 모공이 작고 거칠음
- 주름이 잘 생김

### 지성 (Oily)
- 피부가 윤기가 있고 번들거림
- 모공이 크고 블랙헤드가 많음
- 트러블이 잘 생김

### 복합성 (Combination)
- T존(이마, 코, 턱)은 지성, U존(볼)은 건성
- 모공 크기가 불균형
- 피부 타입 관리가 어려움

### 민감성 (Sensitive)
- 자극에 민감하고 붉어짐
- 가려움, 따가움 증상
- 저자극 제품 필요

## FAQ

### Q: 분석에 얼마나 걸리나요?
A: 일반적으로 1-3분 소요됩니다. 이미지 크기와 서버 상태에 따라 달라질 수 있습니다.

### Q: 분석 결과는 얼마나 정확한가요?
A: AI 모델은 대규모 데이터로 학습되었으나, 100% 정확성을 보장하지 않습니다. 참고용으로 활용하세요.

### Q: 이전 분석 결과를 볼 수 있나요?
A: 네, "분석 이력" 메뉴에서 이전 분석 결과를 확인할 수 있습니다.

### Q: 이미지를 여러 번 업로드할 수 있나요?
A: 네, 제한 없이 업로드할 수 있습니다. 단, 속도 제한이 적용됩니다.

### Q: 분석 결과를 공유할 수 있나요?
A: 현재는 개인 계정에서만 확인 가능합니다. 공유 기능은 추후 추가 예정입니다.

## 트러블슈팅

### 이미지 업로드 실패

**문제:** 이미지 업로드가 계속 실패합니다.

**해결:**
1. 이미지 크기가 20MB 이하인지 확인
2. 지원되는 형식(JPG, JPEG, PNG, WEBP)인지 확인
3. 인터넷 연결 상태 확인
4. 브라우저 캐시 삭제 후 재시도

### 분석이 진행되지 않음

**문제:** 분석 상태가 "pending"에서 변하지 않습니다.

**해결:**
1. 페이지 새로고침
2. 잠시 후 다시 시도
3. 문제 지속 시 고객센터 문의

### 점수가 비정상적으로 높/낮음

**문제:** 점수가 실제 피부 상태와 다릅니다.

**해결:**
1. 이미지 촬영 조건 확인 (조명, 각도)
2. 메이크업 제거 후 재촬영
3. 피부 타입 확인 및 수정
