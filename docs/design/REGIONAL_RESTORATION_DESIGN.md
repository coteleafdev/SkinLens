# 지역 선택 복원 설계 (Regional Restoration Design)

> **문서 버전:** 1.0.0  
> **대상 프로젝트 버전:** 1.0.0  
> **마지막 업데이트:** 2026-05-31  
> **상태:** 활성

이 문서는 특정 얼굴 영역(눈가, 입가 등) 선택적 복원 기능의 기술 설계를 설명합니다.

---

## 1. 개요

### 1.1 현재 상황
- 전체 얼굴 복원만 지원
- 사용자가 원하지 않는 영역까지 복원됨
- 미세 조정 불가

### 1.2 제안된 기능
- 특정 얼굴 영역 선택적 복원
- MediaPipe 랜드마크 기반 영역 정의
- 영역별 복원 강도 조절
- 실시간 프리뷰와 연동

### 1.3 기대 효과
- 미세 조정 가능
- 과도한 복원 방지
- 사용자 만족도 향상

## 3. 기술 설계

### 3.1 마스크 생성
1. MediaPipe Face Mesh에서 랜드마크 추출
2. 선택된 영역에 해당하는 랜드마크 필터링
3. 랜드마크를 연결하여 다각형 생성
4. 다각형 내부를 채워 마스크 이미지 생성
5. 마스크 가장자리 부드럽게 처리 (feathering)

### 3.2 영역별 복원 적용
1. 원본 이미지에 마스크 적용
2. 마스크된 영역만 복원 엔진 실행
3. 복원된 영역을 원본 이미지에 블렌딩
4. 가장자리 자연스러운 전환 (alpha blending)

### 3.3 API 엔드포인트
- **경로:** `/v1/restoration/preview`
- **메서드:** `POST`
- **요청 바디:**
  ```json
  {
    "job_id": "string",
    "restoration_strength": 50,
    "regions": ["eye_area", "mouth_area"]
  }
  ```
- **응답:**
  ```json
  {
    "preview_image_base64": "string",
    "restoration_strength": 50,
    "regions": ["eye_area", "mouth_area"],
    "message": "string"
  }
  ```

### 3.4 클라이언트 UI/UX
- **영역 선택 UI:** 체크박스 또는 토글 버튼으로 영역 선택
- **시각적 피드백:** 선택된 영역을 얼굴 이미지 위에 하이라이트
- **슬라이더 연동:** 영역별 복원 강도 조절 (선택 사항)
- **프리뷰:** 선택된 영역 복원 결과 실시간 프리뷰

## 5. 성능 고려사항

- **마스크 생성:** MediaPipe 랜드마크 추출은 빠르지만, 마스크 생성 시 최적화 필요
- **복원 처리:** 영역이 작을수록 처리 시간 단축
- **캐싱:** (job_id, restoration_strength, regions) 조합으로 캐싱
- **GPU 활용:** 복원 엔진 GPU 가속

---

## 6. 성공 지표

- **마스크 생성 시간:** 0.5초 이내
- **영역별 복원 시간:** 전체 복원 대비 30-50% 단축
- **사용자 만족도:** 기능 도입 후 긍정적 피드백 +20%
- **과도한 복원 불만:** 사용자 불만 감소 -30%
