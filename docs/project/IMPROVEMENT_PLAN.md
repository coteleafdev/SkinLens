## 1. 즉시 개선 필요 (P0) - 완료 ✅

### 1.1 인증 보안 강화
- **위치:** `src/server/routers/auth.py`
- **현황:** 이미 bcrypt 지원 구현됨
- **완료:** `_verify_pw()` 함수가 bcrypt 해시($2로 시작)를 지원
- **운영 가이드:**
  ```bash
  # bcrypt 해시 생성
  python -c "from passlib.context import CryptContext; pwd_context = CryptContext(schemes=['bcrypt'], deprecated='auto'); print(pwd_context.hash('your_password'))"
  
  # 환경변수에 해시된 비밀번호 설정
  export ADMIN_PASSWORD="$2b$12$..."
  export ANALYST_PASSWORD="$2b$12$..."
  ```

### 1.2 로깅 표준화
- **위치:** `src/gui/skin_analysis_pipeline.py`, `src/monitoring/metrics_collector.py`
- **현황:** 완료
- **수정 내용:**
  - `skin_analysis_pipeline.py`: DEBUG, 경고, 오류, 시간 요약 메시지를 logger로 교체 (20+건)
  - `metrics_collector.py`: print_report()의 모든 print()를 logger.info로 교체
- **유지한 것 (사용자 인터페이스):**
  - JSON 출력 (CLI 표준 출력)
  - measurement_report_string (측정 결과)
  - PySide6 ImportError 메시지 (설치 안내)
  - _print_self_help() (도움말)
  - skin_scoring.py의 print_results() (사용자 인터페이스 함수)
