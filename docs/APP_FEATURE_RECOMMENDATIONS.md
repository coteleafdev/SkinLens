# 앱 기능 추천

SkinLens v1.0의 현재 기능을 바탕으로 앱에 추가하면 좋은 기능들을 제안합니다.

---

## 1. 피부 일기 (Skin Diary)

### 개요
사용자의 피부 상태 변화를 시간 순서대로 기록하고 관리하는 기능

### 주요 기능
- **매일 피부 상태 사진 저장**: 일일 피부 셀카 업로드 및 관리
- **피부 점수 변화 그래프**: 시계열 그래프로 피부 점수 추이 시각화
- **사용된 제품 기록**: 사용한 제품과 사용 날짜 기록
- **피부 개선 타임라인**: 전후 비교 및 개선 과정 시각화

### 연동 API
- `GET /v1/enhancements/trends` - 분석 추이 조회
- `POST /v1/enhancements/upload` - 이미지 업로드
- `GET /v1/analysis/jobs/{job_id}` - 분석 결과 조회

### 구현 우선순위
- 높음: 현재 분석 추이 기능과 직접 연동 가능

## 2. AI 피부 코치

### 개요
분석 결과를 기반으로 맞춤형 피부 관리 조언을 제공하는 AI 기능

### 주요 기능
- **분석 결과 기반 맞춤형 조언**: 피부 타입과 문제점에 따른 개인화된 솔루션
- **주간/월간 피부 관리 루틴 추천**: 주기적인 관리 플랜 제안
- **계절별 피부 관리 팁**: 날씨와 계절에 따른 맞춤형 팁
- **생활 습관 개선 제안**: 수면, 식습관 등 생활 습관 개선 조언

### 연동 API
- `GET /v1/analysis/jobs/{job_id}` - 분석 결과 조회
- `GET /v1/customers/{customer_id}` - 고객 정보 조회
- `POST /v1/llm/generate` - LLM 소견 생성

### 구현 우선순위
- 높음: 현재 LLM 소견 기능 확장

## 3. 제품 비교 리뷰

### 개요
사용자가 사용한 제품을 비교하고 효과를 분석하는 기능

### 주요 기능
- **사용자가 사용한 제품 비교**: 여러 제품의 효과 비교
- **전후 사진 비교**: 제품 사용 전후 피부 상태 비교
- **제품별 효과 분석**: 각 제품의 피부 점수 영향 분석
- **재구매 추천**: 사용자 피드백 기반 재구매 추천

### 연동 API
- `GET /v1/orders/products/{product_id}/feedback` - 제품 피드백 조회
- `POST /v1/orders/feedback` - 피드백 등록
- `GET /v1/enhancements/trends` - 분석 추이 조회

### 구현 우선순위
- 중간: 현재 피드백 시스템 확장

## 구현 상세

### ✅ 구현 완료 (4개)

#### 1. 피부 일기 (Skin Diary)
- **DB 테이블**: `skin_diary` (버전 26)
- **API 엔드포인트**:
  - `POST /v1/app/diary` - 피부 일기 엔트리 생성
  - `GET /v1/app/diary/{customer_id}` - 피부 일기 조회
- **기능**: 매일 피부 상태 기록, 점수 변화 추적, 기분/날씨 기록

#### 2. 개인화된 대시보드 (Personalized Dashboard)
- **DB 테이블**: 
  - `customer_goals` (버전 27)
  - `achievements` (버전 28)
  - `customer_achievements` (버전 29)
- **API 엔드포인트**:
  - `POST /v1/app/goals` - 목표 생성
  - `PUT /v1/app/goals/{goal_id}/progress` - 목표 진행률 업데이트
  - `GET /v1/app/goals/{customer_id}` - 목표 조회
  - `POST /v1/app/achievements` - 업적 생성
  - `POST /v1/app/achievements/earn` - 업적 획득
  - `GET /v1/app/achievements/{customer_id}` - 업적 조회
- **기능**: 개인 목표 설정, 업적 시스템, 진행률 추적

#### 3. 제품 구독 서비스 (Product Subscription)
- **DB 테이블**: `product_subscriptions` (버전 30)
- **API 엔드포인트**:
  - `POST /v1/app/subscriptions` - 구독 생성
  - `GET /v1/app/subscriptions/{customer_id}` - 구독 조회
- **기능**: 정기 배송 구독, 자동 재주문

#### 4. 피부 건강 챌린지 (Skin Health Challenge)
- **DB 테이블**:
  - `challenges` (버전 31)
  - `customer_challenges` (버전 32)
- **API 엔드포인트**:
  - `POST /v1/app/challenges` - 챌린지 생성
  - `POST /v1/app/challenges/join` - 챌린지 참여
  - `PUT /v1/app/challenges/progress` - 챌린지 진행률 업데이트
  - `GET /v1/app/challenges/{customer_id}` - 챌린지 조회
- **기능**: 30일 피부 개선 챌린지, 목표 설정 및 추적, 보상 시스템
