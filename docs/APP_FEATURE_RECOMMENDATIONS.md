# 앱 기능 추천 (App Feature Recommendations)

> **문서 버전:** 1.5.0  
> **대상 프로젝트 버전:** 1.0.0  
> **마지막 업데이트:** 2026-06-01  
> **상태:** 활성

SkinLens v1.0의 현재 기능을 바탕으로 앱에 추가하면 좋은 기능들을 제안합니다.

---

## 1. 피부 일기 (Skin Diary)

### 개요
사용자의 피부 상태 변화를 시간 순서대로 기록하고 관리하는 기능

### 주요 기능
- **매일 피부 상태 사진 저장**: 일일 피부 셀카 업로드 및 관리
- **피부 점수 변화 그래프**: 시계열 그래프로 피부 점수 추이 시각화
- **사용된 제품 기록**: 사용한 제품과 사용 날짜 기록
- **피부 개선 타임라인**: 전후 비교 및 개선 과정 시각화

### 연동 API
- `GET /v1/enhancements/trends` - 분석 추이 조회
- `POST /v1/enhancements/upload` - 이미지 업로드
- `GET /v1/analysis/jobs/{job_id}` - 분석 결과 조회

### 구현 우선순위
- 높음: 현재 분석 추이 기능과 직접 연동 가능

---

## 2. PCR 검사 요청 및 결과 보기

### 개요
피부 상태 정밀 분석을 위한 PCR 검사 요청 및 결과 확인 기능

### 주요 기능
- **PCR 검사 요청**: 피부 상태 정밀 분석을 위한 검사 요청
- **PCR 검사 키트 자동 발송**: 검사 요청 시 자동으로 PCR 검사 키트 배송
- **검사 결과 조회**: PCR 검사 결과 확인 및 해석
- **검사 이력 관리**: 과거 검사 이력 및 추이 확인
- **의료 전문가 상담 연계**: 검사 결과 기반 전문가 상담 예약

### 연동 API
- `POST /v1/app/pcr/request` - PCR 검사 요청
- `GET /v1/app/pcr/results/{customer_id}` - 검사 결과 조회
- `GET /v1/app/pcr/history/{customer_id}` - 검사 이력 조회
- `POST /v1/app/pcr/consultation` - 전문가 상담 예약

### 구현 우선순위
- 높음: 반드시 필요한 핵심 기능

## 3. 기성품 구매 (Ready-made Products Purchase)

### 개요
미리 준비된 5종의 기성품을 앱에서 직접 구매하는 기능

### 주요 기능
- **기성품 목록 조회**: 5종의 기성품 목록 및 재고 확인
- **기성품 상세 정보**: 제품 설명, 가격, 재고 수량, 카테고리 정보
- **장바구니 기능**: 여러 기성품을 장바구니에 담기
- **주문 생성**: 기성품 주문 및 결제
- **주문 내역 조회**: 기성품 주문 내역 확인
- **배송 추적**: 기성품 배송 상태 추적

### 연동 API
- `GET /v1/orders/products/ready-made` - 기성품 목록 조회
- `POST /v1/orders` - 기성품 주문 생성
- `GET /v1/orders/customer/{customer_id}` - 주문 내역 조회
- `GET /v1/orders/{order_id}/tracking` - 배송 추적

### 구현 우선순위
- 높음: 앱 기본 기능으로 필수

### 주요 기능
- **분석 결과 기반 맞춤형 조언**: 피부 타입과 문제점에 따른 개인화된 솔루션
- **주간/월간 피부 관리 루틴 추천**: 주기적인 관리 플랜 제안
- **계절별 피부 관리 팁**: 날씨와 계절에 따른 맞춤형 팁
- **생활 습관 개선 제안**: 수면, 식습관 등 생활 습관 개선 조언

### 연동 API
- `GET /v1/analysis/jobs/{job_id}` - 분석 결과 조회
- `GET /v1/customers/{customer_id}` - 고객 정보 조회
- `POST /v1/llm/generate` - LLM 소견 생성

### 구현 우선순위
- 높음: 현재 LLM 소견 기능 확장

## 5. 제품 비교 리뷰

### 개요
고객이 사용한 제품의 전성분이 표시된 사진을 제공하는 경우에만 작동하는 제품 비교 및 효과 분석 기능

### 주요 기능
- **전성분 사진 기반 제품 비교**: 전성분 사진을 제공한 제품들의 효과 비교
- **전후 사진 비교**: 제품 사용 전후 피부 상태 비교
- **제품별 효과 분석**: 각 제품의 피부 점수 영향 분석
- **재구매 추천**: 사용자 피드백 기반 재구매 추천

### 제한 사항
- **전성분 사진 필수**: 고객이 사용한 제품의 전성분이 표시된 사진을 제공해야만 기능 작동
- **사진 검증**: 전성분 정보가 포함된 사진인지 AI 기반 검증 필요

### 연동 API
- `GET /v1/orders/products/{product_id}/feedback` - 제품 피드백 조회
- `POST /v1/orders/feedback` - 피드백 등록
- `GET /v1/enhancements/trends` - 분석 추이 조회
- `POST /v1/app/product/ingredient-photo` - 전성분 사진 업로드 및 검증

### 구현 우선순위
- 중간: 현재 피드백 시스템 확장 + 전성분 사진 검증 기능 필요

## 구현 상세

### ✅ 구현 완료 (6개)

#### 1. 피부 일기 (Skin Diary)
- **DB 테이블**: `skin_diary` (버전 26)
- **API 엔드포인트**:
  - `POST /v1/app/diary` - 피부 일기 엔트리 생성
  - `GET /v1/app/diary/{customer_id}` - 피부 일기 조회
- **기능**: 매일 피부 상태 기록, 점수 변화 추적, 기분/날씨 기록

#### 2. PCR 검사 요청 및 결과 보기 (PCR Test Request & Results)
- **DB 테이블**:
  - `pcr_test_requests` (버전 39)
  - `pcr_test_results` (버전 39)
  - `pcr_consultations` (버전 39)
- **API 엔드포인트**:
  - `POST /v1/app/pcr/request` - PCR 검사 요청 (자동 키트 발송)
  - `GET /v1/app/pcr/results/{customer_id}` - 검사 결과 조회
  - `GET /v1/app/pcr/history/{customer_id}` - 검사 이력 조회
  - `POST /v1/app/pcr/consultation` - 전문가 상담 예약
  - `GET /v1/app/pcr/consultations/{customer_id}` - 상담 예약 목록 조회
- **기능**: PCR 검사 요청, 자동 키트 발송, 결과 조회, 이력 관리, 전문가 상담 연계

#### 3. 기성품 구매 (Ready-made Products Purchase)
- **DB 테이블**: `products` (버전 40, is_ready_made 컬럼 추가)
- **API 엔드포인트**:
  - `GET /v1/orders/products/ready-made` - 기성품 목록 조회
  - `POST /v1/orders` - 기성품 주문 생성
  - `GET /v1/orders/customer/{customer_id}` - 주문 내역 조회
  - `GET /v1/orders/{order_id}/tracking` - 배송 추적
- **기능**: 기성품 목록 조회, 주문 생성, 주문 내역 조회, 배송 추적

#### 4. 개인화된 대시보드 (Personalized Dashboard)
- **DB 테이블**: 
  - `customer_goals` (버전 27)
  - `achievements` (버전 28)
  - `customer_achievements` (버전 29)
- **API 엔드포인트**:
  - `POST /v1/app/goals` - 목표 생성
  - `PUT /v1/app/goals/{goal_id}/progress` - 목표 진행률 업데이트
  - `GET /v1/app/goals/{customer_id}` - 목표 조회
  - `POST /v1/app/achievements` - 업적 생성
  - `POST /v1/app/achievements/earn` - 업적 획득
