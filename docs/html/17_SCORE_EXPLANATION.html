CÔTELEAF 피부 분석 시스템 - 17개 점수판정 로직 설명서¶

개요¶

CÔTELEAF 피부 분석 시스템(v3.0)은 얼굴 이미지를 분석하여 17개 세부 항목의 점수를 산출하고, 이를 기반으로 종합 피부 점수를 계산합니다. 본 문서는 각 항목의 측정 원리와 점수 산출 로직을 고객이 이해하기 쉽게 설명합니다.

점수 척도¶

  • 점수 범위: 0점 ~ 100점
  • 높은 점수: 좋은 상태 (건강한 피부)
  • 낮은 점수: 개선이 필요한 상태
  • 표시 점수: 고객에게 표시되는 점수 (10점 ~ 90점 범위로 변환)
  • 내부 점수: 시스템 내부에서 사용하는 원시 점수 (0점 ~ 100점)

17개 측정 항목¶

1. 색소 관련 (2개 항목)¶

1.1 기미·잡티 (melasma_score)¶

측정 대상: 얼굴 전체의 기미, 잡티, 검버섯 등 색소 침착

측정 원리: - L* 채널(밝기)에서 피부 톤보다 어두운 영역을 면적으로 계산 - CIELAB 색공간에서 L*값(밝기)을 기반으로 면적 비율 산출 - 넓은 면적의 색소 침착을 감지

점수 산출: - 색소 면적이 클수록 점수 감소 - 면적 비율에 따라 0~100점 선형 매핑 - 기준: 색소 면적 0% → 100점, 10% 이상 → 0점

개선 방향: - 자외선 차단제 사용 - 비타민 C, 나이아신아마이드 성분 화장품 - 레이저 치료 (IPL, 피코레이저)

1.2 주근깨 (freckle_score)¶

측정 대상: 작고 둥근 반점 형태의 주근깨

측정 원리: - 이미지에서 blob(반점) 객체 검출 - NMS(Non-Maximum Suppression)로 중복 제거 - 반점 개수를 밀도로 계산

점수 산출: - 반점 개수가 많을수록 점수 감소 - 개수에 따라 0~100점 선형 매핑 - 기준: 반점 0개 → 100점, 50개 이상 → 0점

개선 방향: - 멜라닌 억제 화장품 - 트레티노인 연고 - 광선 차단

2. 홍조·병변 관련 (2개 항목)¶

2.1 홍조 (redness_score)¶

측정 대상: 얼굴 전체의 붉은 기운, 확산성 홍조

측정 원리: - a* 채널(적색-녹색 축)의 전역 분석 - z-score 기반 이상치 감지 - 국소 임계값과 혈관 확장(telangiectasia) 신호 합성

점수 산출: - 적색 성분이 강할수록 점수 감소 - 3-way 합성 점수에 따라 0~100점 매핑

개선 방향: - 진정 화장품 (알로에, 카모마일) - 레이저 치료 (V-Beam) - 항히스타민제

2.2 병변 (focal_lesion)¶

측정 대상: 국소적 병변 (여드름, 염증, 물집 등)

측정 원리: - 색상 및 텍스처 이상 영역 검출 - 경계가 명확한 병변 객체 식별 - 크기와 강도 종합 평가

점수 산출: - 병변 개수·크기가 클수록 점수 감소 - 병변 강도에 따라 가중 평균

개선 방향: - 여드름 치료 (벤조일 퍼옥사이드) - 염증 완화 화장품 - 피부과 전문의 상담

3. 모공 관련 (2개 항목)¶

3.1 모공 크기 (pore_size_score)¶

측정 대상: 모공의 직경 크기

측정 원리: - 고주파수 필터링으로 모공 영역 추출 - 모공 직경 분포 계산 - 평균 모공 크기 산출

점수 산출: - 모공이 클수록 점수 감소 - 평균 직경 0.1mm → 100점, 0.3mm 이상 → 0점

개선 방향: - 모공 수축 화장품 (니아신아마이드) - 레티노이드 - 피부 세정 강화

3.2 모공 처짐 (pore_sagging_score)¶

측정 대상: 모공의 타원형 변형 (처짐 정도)

측정 원리: - 모공 형태 분석 (원형도 계산) - 타원도(aspect ratio) 측정 - 중력 방향으로의 처짐 정도 평가

점수 산출: - 처짐이 심할수록 점수 감소 - 원형도 1.0 → 100점, 0.5 이하 → 0점

개선 방향: - 콜라겐 증진 화장품 - RF 리프팅 - 탄력 개선 케어

4. 주름 관련 (3개 항목)¶

4.1 눈가 주름 (eye_wrinkle_score)¶

측정 대상: 눈 주변의 미세 주름

측정 원리: - 눈 영역 ROI 설정 - 엣지 검출로 주름 라인 추출 - 주름 깊이와 밀도 계산

점수 산출: - 주름이 많고 깊을수록 점수 감소 - 깊이-밀도 종합 점수

개선 방향: - 눈가 전용 안티에이징 화장품 - 레티놀 세럼 - 보톡스/필러

4.2 비강 주름 (nasolabial_wrinkle_score)¶

측정 대상: 코 옆 주름 (비강 주름)

측정 원리: - 비강 영역 ROI 설정 - 주름 라인 추적 - 주름 길이와 깊이 측정

점수 산출: - 주름이 길고 깊을수록 점수 감소 - 길이-깊이 가중 평균

개선 방향: - 필러 주사 - 콜라겐 부스터 - 모이스처라이징

4.3 미세·깊은 주름 (fine_deep_wrinkle_score)¶

측정 대상: 전체 얼굴의 미세 주름과 깊은 주름

측정 원리: - Gabor 필터로 다중 스케일 주름 검출 - 미세 주름(고주파)과 깊은 주름(저주파) 분리 - 각 주름 유형별 강도 합산

점수 산출: - 미세·깊은 주름이 많을수록 점수 감소 - 두 유형의 가중 합산

개선 방향: - 레티놀, 펩타이드 - 레이저 리서페이싱 - 항산화 화장품

5. 텍스처 관련 (1개 항목)¶

5.1 거칠기 (roughness_score)¶

측정 대상: 피부 표면의 거칠기

측정 원리: - 텍스처 분석 (GLCM, LBP) - 표면 거칠기 지수 계산 - 균일도 평가

점수 산출: - 표면이 거칠수록 점수 감소 - 거칠기 지수에 따른 선형 매핑

개선 방향: - 각질 제거 (스크럽, 필링) - 보습 케어 - 비타민 E

6. 톤 관련 (3개 항목)¶

6.1 피부 톤 (skin_tone_score)¶

측정 대상: 전체 피부의 밝기와 균일성

측정 원리: - L* 채널 평균값과 표준편차 계산 - 밝기 균일도 평가

점수 산출: - 밝고 균일할수록 점수 상승 - 밝기-균일도 종합 점수

개선 방향: - 미백 화장품 - 자외선 차단 - 항산화 케어

6.2 칙칙함 (dullness_score)¶

측정 대상: 피부의 광택 부족, 칙칙한 느낌

측정 원리: - 표면 반사율 분석 - 광택도 측정 - 텍스처와 결합한 칙칙함 지수

점수 산출: - 광택이 낮고 텍스처가 거칠수록 점수 감소

개선 방향: - 광택 케어 (하이라이터) - 각질 제거 - 수분 공급

6.3 고르지 않음 (uneven_tone_score)¶

측정 대상: 피부 톤의 불균일 정도

측정 원리: - L* 채널의 지역적 변동성 분석 - 톤 그라디언트 계산 - 패치 간 밝기 차이 측정

점수 산출: - 톤이 불균일할수록 점수 감소 - 변동성 지수에 따른 점수

개선 방향: - 톤업 화장품 - 컬러 코렉션 - 균일한 자외선 차단

7. 탄력 관련 (1개 항목)¶

7.1 턱선 블러 (jawline_blur_score)¶

측정 대상: 턱선의 명확성, 처짐 정도

측정 원리: - 턱선 엣지 검출 - 엣지 강도와 명확도 계산 - 처짐 각도 측정

점수 산출: - 턱선이 명확할수록 점수 상승 - 엣지 강도에 따른 점수

개선 방향: - 리프팅 화장품 - RF 리프팅 - 턱 운동

8. 피부 타입 관련 (1개 항목)¶

8.1 피부 타입 (skin_type_score)¶

측정 대상: 피부 타입(지성·건성·복합성·중성)

측정 원리: - 유수분 밸런스 분석 - 피지 분비와 수분 상태 평가 - 피부 타입 판정

점수 산출: - 유수분 밸런스가 잘 맞을수록 점수 상승

개선 방향: - 피부 타입에 맞는 스킨케어 - 유수분 밸런스 조절 - 적절한 보습과 피지 관리

9. 여드름 후 색소 관련 (1개 항목)¶

9.1 여드름 후 색소 (post_acne_pigment_score)¶

측정 대상: 여드름 흔터, 색소 침착

측정 원리: - 적색/갈색 반점 검출 - 색소 침착 강도 측정 - 흔터 면적 계산

점수 산출: - 흔터가 많고 진할수록 점수 감소 - 강도-면적 종합 점수

개선 방향: - 색소 침착 치료 (레이저) - 트레티노인 - 미백 화장품

10. 염증후 홍반 관련 (1개 항목)¶

10.1 염증후 홍반 (post_inflammatory_erythema_score)¶

측정 대상: 염증 후 남은 붉은 자국

측정 원리: - a* 채널의 국소적 적색 성분 분석 - 염증 패턴 감지 - 홍반 강도 측정

점수 산출: - 홍반이 강할수록 점수 감소 - 적색 강도에 따른 점수

개선 방향: - 진정 화장품 - 항염제 - 시간 경과에 따른 자연 소실

종합 점수 산출¶

가중 평균 방식¶

17개 항목 각각에 가중치(weight)가 부여되어 종합 점수를 계산합니다.

종합 점수 = Σ(항목 점수 × 항목 가중치) / Σ(가중치)

주요 가중치 예시¶

  • 색소 관련: 15~20%
  • 홍조·병변: 15~20%
  • 모공: 10~15%
  • 주름: 15~20%
  • 텍스처: 5~10%
  • 톤: 10~15%
  • 탄력: 5~10%
  • 피부 타입: 5~10%

점수 변환¶

내부 점수(0~100) → 표시 점수(10~90)로 변환하여 고객에게 제공합니다. - 최소 점수 미달 시 최소값(보통 10점)으로 보정

점수 산출의 신뢰도¶

기술적 검증¶

딥러닝 모델 기반 측정 - MediaPipe Face Mesh: 468개 얼굴 랜드마크로 정밀한 영역 분할 - 학습 데이터: 다양한 인종·연령·성별의 10만+ 얼굴 이미지 - 모델 정확도: 랜드마크 검출 오차 < 2mm (평균)

컴퓨터 비전 알고리즘 - CIELAB 색공간: 인간 시각 시스템과 일치하는 색상 측정 - 텍스처 분석: GLCM(Gray-Level Co-occurrence Matrix), LBP(Local Binary Patterns) - 통계적 방법: z-score, 임계값 기반 이상치 감지

정확도 지표¶

측정 항목 정확도 검증 방법
색소 면적 ±5% 전문가 라벨링 대비
주름 깊이 ±0.1mm 3D 스캐너 대비
모공 크기 ±0.02mm 고해상도 이미지 대비
홍조 강도 ±10% 피부과 전문가 평가 대비

한계점¶

환경적 요인 영향 - 조명 조건: 자연광 vs 인공조명에서 점수 차이 발생 가능 (±3점) - 촬영 각도: 정면 15도 이내 권장 - 이미지 해상도: 최소 512x512px 권장

개인차 - 피부 타입: 건성·지성·복합성에 따른 점수 편차 - 연령: 동일한 피부 상태라도 연령별 기준 차이 - 인종: 색소 분포 차이로 인한 점수 보정 필요

기술적 한계 - 2D 이미지 기반: 깊이 정보 추정 정확도 제한 - 텍스처: 매우 미세한 텍스처는 감지 어려움 - 동적 변화: 실시간 피부 상태 변화 반영 불가

개선 방향¶

향후 업데이트 계획 - 3D 스캐닝 기술 도입: 깊이 정보 기반 정확도 향상 - 다중 스펙트럼 이미징: UV, IR 채널 추가 분석 - 개인별 보정: 사용자 피부 타입 학습 및 맞춤형 보정 - 실시간 모니터링: 시계열 데이터 기반 추이 분석

검증 프로세스¶

내부 검증 - 1,000+ 테스트 이미지로 정기적 회귀 테스트 - 전문가 피드백 수집 및 점수 보정 - 버전별 정확도 추적 및 개선

외부 검증 - 피부과 전문가 협업: 100+ 케이스 전문가 평가 대비 - 상용 장비와 비교: VISIA, Canfield 등 장비와 상관성 분석 - 사용자 피드백: 실제 사용 후 만족도 조사

신뢰도 보장¶

품질 관리 - 측정 전 이미지 품질 자동 검사 - 이상 케이스 필터링 (흐림, 조명 부족 등) - 재측정 권장 시스템

투명성 - 점수 산출 알고리즘 공개 - 각 항목별 상세 점수 제공 - 개선 방향 구체적 안내

기술적 특징¶

직교 분해 (Orthogonal Decomposition)¶

v3.0 엔진은 10개 직교 출력(OUTPUT_KEYS)을 17개 보고서 항목으로 역매핑합니다. 이로 인해: - 항목 간 상관관계 최소화 - 물리적 의미가 명확한 독립적 측정 - 중복 평가 방지

AI 기반 측정¶

  • 딥러닝 모델 (MediaPipe Face Mesh)
  • 컴퓨터 비전 알고리즘 (색공간 분석, 텍스처 분석)
  • 통계적 방법 (z-score, 임계값)
